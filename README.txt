TIC-TAC-TOE

AUTHOR: 
Dhruv Bhaskar

LAST MODIFIED: 
1st January 2014

ABOUT:
This is a simple Tic-Tac-Toe console game written in C.
The game can operate on two difficulty levels, easy and advanced.

In the easy level, the computer chooses its moves randomly whereas in the advanced mode,
the computer actively tries to prevent the user from winning.

USAGE:
-> Enter 'make' in command prompt.
-> Enter './output easy' to play in the easy mode or enter './output advanced' to play in advanced mode.
